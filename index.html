Bonjour! Je m'apelle Guillaume Touzin, un enseignant

je vous invite à visiter ma page Kaggle et pour voir mes performance dans les compétitions aux quelles j'ai
participées et vous pouvez consulter mon répertoire GitHub pour le code source de certain projets.
