Data science projects

Here's a list of fun projects I worked on as a way to learn some new aspect of data science. I hope you'll have fun reading those articles and that they will help you learn something new.

  • IMDB 5000 movies data set
    • Cleaning a data set of 5000 films from the website IMDB.
    • Data cleaning
  • "Ghouls, Goblins, and Ghosts... Boo!" data set
    • For the Kaggle competition Ghouls, Goblins, and Ghosts... Boo! I had to classifie observations in three different cathegories. In this article, we see the efficiency of different clustering algorithm for the classification of non-linearly separable data.
    • Data exploration, data engineering, data modelisation, data visualisation, cluster algorithm, classification algorithm
  • Titanic data set
    • I entered the competition Titanic: Machine Learning from Disaster on the site Kaggle.com. In this competiton , I had to use information on the passenger to predict if they were likely to survive the sinking of the Titanic.
    • Data exploration, data engineering
